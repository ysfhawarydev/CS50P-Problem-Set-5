# 🐍 CS50 Python - Problem Set 5 Solutions

This repository contains my solutions for **Problem Set 5** from **CS50's Introduction to Programming with Python**.

The problem set focuses on writing automated tests, validating program behavior, identifying bugs, and improving code reliability using the `pytest` framework.

---

## 📚 Problems Included

### 🧪 Testing My Twttr

Write and test a function that removes vowels from text.

### ⛽ Testing Fuel

Create tests for functions that calculate and display fuel percentages.

### 📅 Testing Seasons

Verify date calculations and validate program outputs.

### 🎓 Testing Project Functions

Practice building comprehensive test suites for Python programs.
